# WASTE-CLASSIFICATION-PROJECT

# Description

This project is a deep learning-based image classification system that classifies waste like recyclable and organic using Transfer Learning (VGG16). It demonstrates how pretrained models can be used for real-world problems.

# Features
Image classification using CNN
Transfer Learning with VGG16
Training & validation accuracy visualization
Dummy dataset (can be replaced with a real dataset)

# Tech Stack
Python
TensorFlow / Keras
NumPy
Matplotlib
Scikit-learn
